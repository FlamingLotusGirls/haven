# Haven button node - Toaster

We're catching the disconnect when the toast pops in 80's era toaster.

This information is sent to the trigger gateway where it can be fanned out
to the rest of the system for operation.

DeviceTrigger.h is a standard library for defining triggers and sending them
to the trigger gateway.

For board reference, the boards we are using seem to be clones of [NodeMCU 32S]
(https://www.espboards.dev/esp32/nodemcu-32s/).
